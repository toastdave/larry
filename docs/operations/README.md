# Operations Runbooks

These runbooks cover the MVP support path for Larry, including the first hosted threshold checks and alert hooks.

- `local-development.md` - bring the stack up, inspect services, and recover local Docker workflows
- `provider-outages.md` - triage AI, search, and sports-data outages with environment-specific checks
- `billing-reconciliation.md` - resolve checkout, webhook, and entitlement mismatches
- `alerts-and-dashboards.md` - dashboard slices and alert conditions worth wiring first
